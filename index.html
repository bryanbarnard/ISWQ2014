ISWQ2014 - Hypermedia API Documentation

This is documentation for the sample movie API created for my Independent Study Winter Quarter 2014 at DePaul University

API Details

Endpoint:

  • NODE.JS Implementation: http://iswq2014.bryanbarnard.com:1337/api/
  • Python Implementation: http://iswq2014.bryanbarnard.com:1338/api/

Supported Content Types:

  • application/vnd.collection+json: Collection+JSON Hypermedia Content Type
  • application/json: JSON provided for ease of use in browser, not a Hypermedia Content Type

Operations

  • GET Get Billboard /api/ 200
  • GET Get Movie Resource Collection /api/movies 200
  • POST Create Movie Resource /api/movies 200
  • GET Get Movie Resource /api/movies/{movieid} 201
  • PUT Modify Movie Resource /api/movies/{movieid} 200
  • DELETE Delete Movie Resource /api/movies/{movieid} 204

Sample CURL Requests - NODEJS API

curl -v http://iswq2014.bryanbarnard.com:1337/api/ \
                -X GET \
                -H 'Connection: close' \
                -H 'Accept: application/vnd.collection+json'
            

curl -v http://iswq2014.bryanbarnard.com:1337/api/movies \
                -X GET \
                -H 'Connection: close' \
                -H 'Accept: application/vnd.collection+json'
            

curl -v http://iswq2014.bryanbarnard.com:1337/api/movies?name={movieName} \
                -X GET \
                -H 'Connection: close' \
                -H 'Accept: application/vnd.collection+json'
            

curl -v http://iswq2014.bryanbarnard.com:1337/api/movies/{movieid} \
                -X GET \
                -H 'Connection: close' \
                -H 'Accept: application/vnd.collection+json'
            

curl -v http://iswq2014.bryanbarnard.com:1337/api/movies \
                -X POST \
                -H 'Connection: close' \
                -H 'Accept: application/vnd.collection+json' \
                -H 'Content-Type: application/vnd.collection+json' \
                -d '{ "template": { "data": [ { "name": "name", "value": "Fletch" }, { "name": "description", "value": "Chevy Chase as worlds best detective." }, { "name": "datePublished", "value": "1986-03-28T21:51:08.406Z" }, { "name": "about", "value": "Cheve Chase as LA Detective" }, { "name": "genre", "value": "comedy" }, { "name": "version", "value": "1.0" }, { "name": "timeRequired", "value": "PT100M" }, { "name": "contentRating", "value": "r" } ] } }'
            

curl -v http://iswq2014.bryanbarnard.com:1337/api/movies/{movieid}
                -X PUT \
                -H 'Connection: close' \
                -H 'Accept: application/vnd.collection+json' \
                -H 'Content-Type: application/vnd.collection+json' \
                -d '{ "template": { "data": [ { "name": "name", "value": "Fletch Lives" }, { "name": "description", "value": "Fletch, again" }, { "name": "datePublished", "value": "1992-03-28T21:51:08.406Z" }, { "name": "about", "value": "Chevy Chase as LA Detective" }, { "name": "genre", "value": "comedy" }, { "name": "version", "value": "1.0" }, { "name": "timeRequired", "value": "PT140M" }, { "name": "contentRating", "value": "r" } ] } }'
            

curl -v http://iswq2014.bryanbarnard.com:1337/api/movies/{movieid}
                -X DELETE \
                -H 'Connection: close' \
                -H Cache-Control:no-cache \
